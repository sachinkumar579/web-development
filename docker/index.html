• WSL installation
  • Run command in powershell as admin : wsl --install
  • PS C:\WINDOWS\system32> wsl --install
    • Console statements getting printed :
    • Installing: Virtual Machine Platform Virtual Machine Platform has been installed. Installing: Windows Subsystem for Linux Windows Subsystem for Linux has been installed. Downloading: WSL Kernel Installing: WSL Kernel WSL Kernel has been installed. Downloading: Ubuntu The requested operation is successful. Changes will not be effective until the system is rebooted. PS C:\WINDOWS\system32> Restart your system The default distribution is Ubuntu . So open Ubuntu from Windows Start Menu Create an account .
    • Check option Windows Subsystem for Linux : Go to control panel - Turn on features - check option WSL Install Docker Desktop
Docker desktop installation
  • Open Docker Desktop after installation . Run getting started image from command prompt .
  • Launch from browser and check running container status in Docker desktop
